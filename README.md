# anti-pattern-ai-platform
## Project Description
This project uses Machine Learning and Deep Learning to detect software design anti-patterns (such as God Class and Feature Envy) and suggests refactoring solutions through a user platform.

## Features
- Anti-pattern detection using ML/DL
- Code metrics extraction
- Refactoring suggestions
- Web platform (future work)
  
  ## Tech Stack
- Python
- Scikit-learn
- Deep Learning (TensorFlow/PyTorch)
- FastAPI
- React (future)
